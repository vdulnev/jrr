# JRR — JRiver Remote

Remote control application for [JRiver Media Center](https://jriver.com/) via the MCWS (Media Center Web Service) API.

## What it does

- Control audio playback: play, pause, stop, skip, seek, volume
- View now-playing info with artwork and audio details
- Manage the Playing Now queue
- Switch between and link playback zones
- Works across mobile and desktop platforms

## Project structure

- `spec.md` — language-agnostic specification that all implementations follow
